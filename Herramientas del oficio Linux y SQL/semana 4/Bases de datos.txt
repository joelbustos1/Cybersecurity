Una base de datos relacional se estructura con tablas relacionadas entre sí. Veamos más sobre la base de datos relacional. Empecemos con una tabla individual en una base de datos grande con información de una empresa. Cada tabla tiene campos de información.
Las columnas que relacionan dos tablas entre sí se llaman claves. Hay dos tipos de claves. La primera es la clave primaria. La clave primaria es una columna en la que cada fila tiene una entrada única. No debe tener valores duplicados, valores nulos ni valores vacíos. Con la clave primaria, identificamos de forma única cada fila de la tabla.
La segunda es la clave foránea, o externa. Es una columna en una tabla que es una clave primaria en otra tabla. A diferencia de la primaria, la foránea puede tener valores vacíos y duplicados. Con la clave foránea, conectamos dos tablas.
